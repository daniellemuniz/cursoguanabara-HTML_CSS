Personalizando Links

Peseudo-classe: Visited (Muda a cor se o site for visitado); hover( passar o mouse por cima e mudar); active ( mundar se clicar em cima).
As pseudo-classes definem estados de elementos dentro do HTML.
para usar as pseudo-classes: tag:{}

Os pseudo-elementos mexem no conteúdo daquele elemento. Para usar os pseudos-elementos: tag::{}
Os pseudos-elementos são: after( coloca algo depois do conteúdo); before(coloca algo antes do conteúdo);
