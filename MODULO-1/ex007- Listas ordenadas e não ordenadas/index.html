Trabalhando com Listas

Listas ordenadas (a ordem faz diferença)

Primeiro criar uma tag OL para delimitar que aquilo é uma lista, depois cada item da lista entra dentro da tag LI.

  1. Acordar
  2. tomar café
  3. Fazer comida
  4. Lavar roupa

A tag OL pode receber parametros, esse é o TYPE (1, A, a, I, i)
também é possivel indicar o inicio da contagem usando o paametro START

  1. Acordar
  2. Tomar café
  3. fazer comida
  4. Lavar roupa

Listas não ordenadas (a ordem não importa)

Para listas não ordenadas a tag é UL para delimitar que é uma lista. Depois cada item dentro de uma LI

  • pão
  • Leite
  • Arroz
  • feijão

A tag UL também pode receber parametros TYPE (disc, circle, square) tudo em minusculo

  • pão
  • leite
  • arroz
  • feijão
